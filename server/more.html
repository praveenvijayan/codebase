Styledocs

I. Installation

  1. Install Nodejs
  2. Install Styledocs

Node Installation

Download node from nodejs.org and install.

Node styledocs

open command prompt
c:\> npm install -g styledocs

II. Usage

open command prompt
styledocs -n "Name of the project" location-of-the-stylesheet
Example:
c:\far002> styledocs -n "yas water worls" style.css

Build file

  1. Install Ant build tool
  2. Configure environment variables
  3. Download build.xml file

Usage

Copy build.xml file in to your project folder.
Run following comment to initialize a project.
ant -Ddir=priject-name -Dver=jquery-version
Example:* c:\works> ant -Ddir=fli005 -Dver=1.6.4
Assuming project folder is under c:\works\
